# nlp-final


## Run the code

1. Install dependencies

    `pip install -r requirements.txt`

2. Download the dataset

    `python parse_data.py`

3. Run our experiment

    `jupyter lab`

    This will launch a jupyter server in your browser. Open `embedding.ipynb`, then click the ‚è© button at the top of the browser. This will our experiment. You can also run our notebook in VSCode with the Jupyter extension

    Our experiments sometimes produce errors. If they do, just run it again from the beginning.
